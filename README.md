### **사람으로 사는 게 피곤한 당신, 그럼 동물로 변해서 놀아봐요.**

저희 주토피아에서는 

1. **동물만 출입 가능합니다**
2. **여유롭고 유쾌한 분위기를 지향합니다** 
3. **남을 헐뜯고 비방하는 행위는 금지됩니다.**

**주토피아 안에서 다른 동물들과 교감하며 하루의 스트레스를 풀어보세요**
